! Wolf Honore
! Problem 020
! Description: Find the sum of the digits in the number 100!
! Method: Computes 100! and sums the digits by converting to a string and
! mapping each digit back to a number.
! Running time: 0.000275121 seconds

USING: math math.factorials math.parser sequences strings
tools.time ;
IN: eul.p020

: fact-digit-sum ( n -- m )
    n! number>string
    [ 1string string>number ] [ + ] map-reduce ;

: p020-run ( -- sum ) 100 fact-digit-sum ;

: p020-time ( -- sum ) [ p020-run ] time ;

! Wolf Honore
! Problem 001
! Description: Find the sum of all the multiples of 3 or 5 below 1000.
! Method: Generates a list from 1 to 999 and sums every value that is a 
! multiple of 3 or 5. Any non multiple is replaced with a 0.
! Running time: 0.000177331 seconds 

USING: kernel math math.ranges sequences tools.time ;
IN: eul.p001

: mult3or5? ( x -- ? )
    dup [ 3 mod 0 = ] [ 5 mod 0 = ] bi or
        [ ]
        [ drop 0 ] 
    if ;

: sum3or5 ( n -- sum ) [1,b] [ mult3or5? ] map sum ;

: p001-run ( -- sum ) 999 sum3or5 ;
    
: p001-time ( -- time ) [ p001-run ] time ;

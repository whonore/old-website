! Wolf Honore
! Problem 016
! Description: What is the sum of the digits of the number 2^1000?
! Method: Computes 2^1000 and sums the digits by converting to a string and
! mapping each digit back to a number.
! Running time: 0.000362932 seconds

USING: kernel math math.functions math.parser sequences strings
tools.time ;
IN: eul.p016

: pow-2-digit-sum ( pow -- sum ) 
    2^ number>string
    [ 1string string>number ] [ + ] map-reduce ;

: p016-run ( -- sum ) 1000 pow-2-digit-sum ;

: p016-time ( -- sum ) [ p016-run ] time ;
